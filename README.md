# 🤖 SmartBook Telegram Integration System

A full backend system that receives Telegram messages, filters them, and sends them to a main SmartBook system via API.

---

## 🚀 Features

* 📩 Receive Telegram messages in real-time
* 📞 Filter allowed and blocked numbers
* 🧠 Process and validate message content
* 🌐 Send data to external SmartBook API
* 🖥️ Dashboard for monitoring messages
* 🔐 Authentication with SmartBook system
* 📊 Logging and statistics system

---

## 🧠 How It Works

1. Telegram messages are received
2. Numbers are filtered (allowed / blocked)
3. Messages are processed and validated
4. Data is sent to SmartBook API
5. Results are displayed in dashboard

---

## ⚙️ Tech Stack

* Python (Backend)
* Flask (Dashboard API)
* Telethon (Telegram API)
* REST API Integration
* JSON (Data storage)

---

## ▶️ Run the Project

```bash
pip install -r requirements.txt
python launcher.py
```

---

## 🔐 Notes

* Do not expose `.env` or tokens
* Configure API keys before running
