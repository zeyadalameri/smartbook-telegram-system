
# 🤖 SmartBook Telegram Integration System

A backend system that receives Telegram messages, filters them, and sends them to SmartBook API with a monitoring dashboard.

---

## 🚀 Features

* 📩 Receive Telegram messages in real-time
* 📞 Filter allowed and blocked numbers
* 🧠 Process and validate message content
* 🌐 Send data to SmartBook API
* 🖥️ Dashboard for monitoring messages
* 🔐 SmartBook authentication system
* 📊 Logging and statistics

---

## 🧠 How It Works

1. Telegram messages are received
2. Numbers are filtered (allowed / blocked)
3. Messages are processed
4. Data is sent to SmartBook API
5. Results are shown in dashboard

---

## ⚙️ Tech Stack

* Python
* Flask
* Telethon
* REST APIs
* JSON Storage

---

## ▶️ Run the Project

```bash
pip install -r requirements.txt
python launcher.py
```
